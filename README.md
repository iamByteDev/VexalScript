# Vexal v10.2 Documentation

## Quick Start (CLI)

Run the Vexal compiler or interpreter using Node.js:

```bash
# Run the Vexal Environment
Clone this git repo
Run:
node vexal-main.js
or
bun run vexal-main.js
or
npm run vexal-main.js
or
whatever pacakge manager

# Follow the prompts to:
# 1. Run a specific file
# 2. Enter REPL (Interactive Mode)
# 3. Transcribe (Compile) .vx files to .js
```

# Full docs in the REPL command

```bash
vx
```
