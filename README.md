# Vexal v10.2 Documentation

## Quick Start (CLI)

Run the Vexal compiler or interpreter using Node.js:

```bash
# Run the Vexal Environment
Clone this git repo
Run:
node vexal-main.js
or
bun run vexal-main.js
or
npm run vexal-main.js
or
whatever pacakge manager

# Follow the prompts to:
# 1. Run a specific file
# 2. Enter REPL (Interactive Mode)
# 3. Transcribe (Compile) .vx files to .js
```

# Full docs in the REPL command

```bash
# After running the program with Bun, NPM etc, close the dialog asking if you want to open a file, then type:
vx
```

# Basic knowledge: .vx files

You can run .vx files with VexalScript inside. You eed to get the filepath for the .vx file, then paste it when prompted upon startup, or run:

```bash
openfile
# Run in REPL
```
